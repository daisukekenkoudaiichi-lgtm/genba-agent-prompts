# 人事自動化 プロンプト集

人事・採用・労務業務を Claude Code で自動化するプロンプト群です。

---

## 収録予定のプロンプト

| ファイル名 | 内容 | ステータス |
|---|---|---|
| `01_採用票作成.md` | 求人票・JD の自動生成 | 準備中 |
| `02_面接評価シート.md` | 面接スコアリングシートの生成 | 準備中 |
| `03_勤怠集計.md` | 勤怠データの集計・レポート化 | 準備中 |
| `04_入社手続きチェックリスト.md` | 入社時の必要書類・手続き一覧 | 準備中 |
| `05_評価フィードバック.md` | 人事評価のフィードバック文生成 | 準備中 |

---

## 使い方（共通）

1. 該当する `.md` ファイルを開く
2. **このプロンプトをコピペして使用** — Claude Code に貼り付ける
3. `[入力]` 箇所を実際の情報に置き換えて実行
