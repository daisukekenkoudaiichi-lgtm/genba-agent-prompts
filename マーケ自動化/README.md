# マーケ自動化 プロンプト集

マーケティング・集客・コンテンツ制作を Claude Code で自動化するプロンプト群です。

---

## 収録予定のプロンプト

| ファイル名 | 内容 | ステータス |
|---|---|---|
| `01_SNS投稿文.md` | X(Twitter)/Instagram/LINE 投稿文の自動生成 | 準備中 |
| `02_メルマガ.md` | メールマガジン本文の生成 | 準備中 |
| `03_広告コピー.md` | LP・バナー広告のコピーライティング | 準備中 |
| `04_SEO記事.md` | SEO を意識したブログ記事の生成 | 準備中 |
| `05_競合分析.md` | 競合サイトの分析レポート生成 | 準備中 |

---

## 使い方（共通）

1. 該当する `.md` ファイルを開く
2. **このプロンプトをコピペして使用** — Claude Code に貼り付ける
3. `[入力]` 箇所を実際の情報に置き換えて実行

---

## SNS別 クイックプロンプト

### X（Twitter）

```
以下の内容で X 投稿文を5パターン作成してください。
テーマ: [入力]
トーン: [プロフェッショナル / カジュアル / 共感型]
ハッシュタグ: 3〜5個含める
文字数: 140字以内
```

### Instagram

```
以下の内容で Instagram キャプション を3パターン作成してください。
テーマ: [入力]
ハッシュタグ: 10〜15個
改行・絵文字を効果的に使用すること
```
