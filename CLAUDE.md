# genba-agent-prompts — Claude Code プロンプト集

## 概要

このリポジトリは、現場業務で即使える **Claude Code 用プロンプト集** です。
コピペするだけで、HP作成・人事・マーケティングの自動化タスクを Claude に任せられます。

---

## フォルダ構成と説明

| フォルダ | 内容 |
|---|---|
| `HP作成/` | ランディングページ・コーポレートサイトの構築プロンプト |
| `人事自動化/` | 採用・勤怠・評価など人事業務の自動化プロンプト |
| `マーケ自動化/` | SNS投稿・メルマガ・広告文など마케팅業務の自動化プロンプト |

---

## 使い方ガイド

### 基本的な使い方

1. 目的のフォルダを開く
2. 該当する `.md` ファイルのプロンプトを確認する
3. **このプロンプトをコピペして使用** — Claude Code のチャット欄に貼り付けて実行

### Claude Code での実行方法

```bash
# Claude Code を起動
claude

# または特定ディレクトリで起動
claude --project /path/to/your/project
```

プロンプトをそのまま貼り付けると、Claude が自動的にタスクを実行します。

---

## ワークフロー例（HP作成）

```
HP作成/01_セットアップ.md   → プロジェクト初期化
HP作成/02_HTML自動生成.md  → LP本体の自動生成
HP作成/03_コマンド化.md    → 繰り返し使えるコマンド化
HP作成/WORKFLOW.md         → 全体の流れ
```

---

## 注意事項

- プロンプトは用途に応じてカスタマイズして使用してください
- `CLAUDE.md` をプロジェクトルートに置くと、Claude が自動で読み込みます
- 機密情報（APIキー等）はプロンプト内に含めないでください
