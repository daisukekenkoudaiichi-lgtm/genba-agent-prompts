# 03 — コマンド化（再利用可能なスラッシュコマンド）

## このプロンプトをコピペして使用

```
このプロジェクトで行った以下の作業を、再利用可能な Claude Code スラッシュコマンドとして保存してください。

【コマンド化したい作業】
1. LP の HTML/CSS 全体生成
2. セクション単体の追加（例: FAQ を追加）
3. カラーテーマの一括変更
4. Lighthouse 改善（パフォーマンス・アクセシビリティ）

【保存場所】
.claude/commands/ フォルダ内に以下のファイルを作成：
- generate-lp.md
- add-section.md
- change-theme.md
- fix-lighthouse.md

各ファイルには、プロンプトテンプレートと使用例を記載してください。
```

---

## スラッシュコマンドの使い方

Claude Code でコマンドを作成すると、`/` で呼び出せます：

```bash
# コマンド一覧を確認
/help

# カスタムコマンドを実行
/generate-lp
/add-section
/change-theme
```

---

## コマンドファイルの書き方

`.claude/commands/generate-lp.md` の例：

```markdown
# LP生成コマンド

$ARGUMENTS を参考に、LP の index.html を生成してください。

【固定設定】
- 技術スタック: HTML/CSS/JS のみ
- フォント: Noto Sans JP
- スマホ対応: 必須
```

---
