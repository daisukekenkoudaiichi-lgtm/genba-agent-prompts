# 02 — HTML自動生成

## このプロンプトをコピペして使用

```
以下の仕様でランディングページ（LP）の HTML/CSS を完全に生成してください。

【クライアント情報】
- 名前/屋号: [入力]
- 提供サービス: [入力]
- 強み・差別化ポイント: [入力]
- ターゲット顧客: [入力]
- CTA（行動喚起）: [例: 無料相談の申し込み / LINE追加 / 資料請求]

【LP構成（セクション順）】
1. ヘッダー（ロゴ + ナビ）
2. FV（ファーストビュー）— キャッチコピー + サブコピー + CTAボタン
3. 課題提起（ターゲットの悩み共感）
4. サービス紹介（特徴3点）
5. 実績・信頼性（数字・事例）
6. お客様の声（3件）
7. よくある質問（FAQ）
8. 料金・プラン
9. 問い合わせフォーム
10. フッター

【デザイン要件】
- カラー: [メインカラーを入力 例: #2563EB（青）]
- フォント: Noto Sans JP（Google Fonts）
- スマホ対応: 必須（モバイルファースト）
- アニメーション: スクロールで要素がフェードイン

【コーディング規約】
- 純粋な HTML / CSS / JS のみ（フレームワーク不使用）
- CSS は style タグ内に記述（外部ファイル不要）
- 画像は unsplash の URL またはプレースホルダーを使用

すべてを index.html 1ファイルにまとめて出力してください。
```

---

## バリエーション

### シンプル版（1スクロール完結）

```
シンプルな1ページLPを作成してください。
セクション: FV → サービス紹介 → 問い合わせ のみ。
```

### リッチ版（アニメーション重視）

```
上記の構成に加えて、以下のアニメーションを追加してください：
- パーティクル背景（FV）
- カウントアップアニメーション（実績数値）
- スムーススクロール
- ハンバーガーメニュー（SP）
```

---

## 実行後の確認

- [ ] 全セクションが揃っている
- [ ] スマホ表示が崩れていない
- [ ] CTAボタンが目立つ位置にある
- [ ] フォームのバリデーションが機能する
