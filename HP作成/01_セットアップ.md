# 01 — プロジェクトセットアップ

## このプロンプトをコピペして使用

```
以下の構成でLPプロジェクトをセットアップしてください。

【プロジェクト情報】
- クライアント名: [クライアント名を入力]
- サービス名: [サービス名を入力]
- ターゲット: [ターゲット層を入力]

【作業内容】
1. 以下のフォルダ・ファイルを作成してください：
   - index.html
   - style.css（または css/style.css）
   - script.js（または js/script.js）
   - CLAUDE.md（このプロジェクト専用の指示書）

2. CLAUDE.md には以下を記載：
   - プロジェクト概要
   - デザイン方針（カラー・フォント・トーン）
   - コーディング規約

3. index.html の基本構造（<!DOCTYPE html> から</html>）を作成

技術スタック: 純粋な HTML / CSS / JavaScript のみ（フレームワーク不使用）
```

---

## カスタマイズポイント

| 項目 | 変更箇所 |
|---|---|
| クライアント名 | プロンプト内 `[クライアント名を入力]` |
| サービス名 | プロンプト内 `[サービス名を入力]` |
| ターゲット層 | プロンプト内 `[ターゲット層を入力]` |

---

## 実行後の確認

- [ ] `index.html` が作成された
- [ ] `CLAUDE.md` に適切な指示が書かれている
