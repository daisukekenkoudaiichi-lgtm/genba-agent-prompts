# HP作成 — 全体ワークフロー

## ステップ概要

```
Step 1: セットアップ         → 01_セットアップ.md
Step 2: HTML自動生成         → 02_HTML自動生成.md
Step 3: コマンド化・再利用   → 03_コマンド化.md
```

---

## Step 1: セットアップ

プロジェクトフォルダを作成し、CLAUDE.md（指示書）を配置する。

**所要時間**: 5〜10分

```bash
mkdir my-lp && cd my-lp
claude
```

→ `01_セットアップ.md` のプロンプトを貼り付ける

---

## Step 2: HTML自動生成

Claude に LP の HTML/CSS/JS を一括生成させる。

**所要時間**: 10〜30分（規模による）

→ `02_HTML自動生成.md` のプロンプトを貼り付ける

---

## Step 3: コマンド化

完成したワークフローをカスタムコマンドとして保存し、次回以降はワンコマンドで再現できるようにする。

**所要時間**: 5分

→ `03_コマンド化.md` のプロンプトを貼り付ける
