# 04 — 佐瀬 LP 自動化エージェント化

## Bash エイリアス登録

```bash
# ~/.bashrc に以下を追加
alias 佐瀬='function _sasedaisuke_agent() {
  cd ~/Github/sasedaisuke-LP
  claude code "
  CLAUDE.md + SKILL.md + progress.md + task.md + KNOWLEDGE.md を読み込んで
  
  以下の修正を実行してください：
  $*
  
  修正後：
  - 確認画面（Yes / 修正 / Cancel）
  - Yes → 自動 push
  - KNOWLEDGE.md に学習メモ記録
  - GitHub Desktop で main にマージ
  - Netlify 自動デプロイ
  "
}; _sasedaisuke_agent'
```

## 使い方

```bash
佐瀬 "プロフィール写真を大きくして"
佐瀬 "お知らせセクションを追加"
佐瀬 "Lighthouse スコア を 95 にして"
```

【動作フロー】
```
佐瀬 "修正内容"
    ↓
Claude Code が CLAUDE.md / SKILL.md を読み込み
    ↓
自動で HTML 修正
    ↓
確認画面（Yes / 修正 / Cancel）
    ↓
Yes → git add / commit / push
    ↓
KNOWLEDGE.md に記録
    ↓
GitHub Desktop で自動マージ
    ↓
Netlify が自動デプロイ
    ↓
完成！
```
