# 04 — 佐瀬 LP 自動化エージェント化

## Bash エイリアス登録

```bash
# ~/.bashrc に以下を追加
alias 佐瀬='function _sasedaisuke_agent() {
  cd ~/Github/sasedaisuke-LP
  claude code "
  CLAUDE.md + SKILL.md + progress.md + task.md + KNOWLEDGE.md を読み込んで
  
  以下の修正を実行してください：
  $*
  
  【修正フロー】
  1. 修正内容を分析
  2. HTML を自動修正（確認なし）
  3. Lighthouse スコア・SEO・モバイル確認
  4. 修正結果をレポート表示
  
  【レポート内容】
  - 修正箇所
  - Lighthouse スコア（Performance / Accessibility）
  - SEO チェック結果
  - モバイル表示確認
  - ファイル変更一覧
  
  5. ユーザーが「修正結果を確認」後に選択
     [Yes] → 自動 push + デプロイ
     [修正] → 修正内容を再指示
     [Cancel] → 中止
  
  Yes を選ぶ場合：
  - 自動で git add / commit / push
  - KNOWLEDGE.md に学習メモ記録
  - GitHub Desktop で main にマージ
  - Netlify 自動デプロイ
  "
}; _sasedaisuke_agent'
```

## 使い方

```bash
佐瀬 "プロフィール写真を大きくして"
    ↓
【自動実行】
Claude が修正内容を分析
HTML を自動修正
Lighthouse / SEO / モバイル確認
    ↓
【修正結果レポート表示】
✅ Lighthouse: Performance 92 / Accessibility 96
✅ SEO: meta description 対応
✅ モバイル: iPhone SE で崩れなし
修正ファイル: index.html
    ↓
【最終確認】
修正結果を確認してから選択
[1] Yes     → そのまま push
[2] 修正    → 別の修正を追加
[3] Cancel  → 中止
```

【動作フロー】
```
佐瀬 "修正内容"
    ↓
Claude が「確認なし」で自動実行：
  - HTML を修正
  - Lighthouse を測定
  - SEO をチェック
  - モバイル確認
    ↓
修正結果レポート表示
    ↓
【ユーザーが確認】
    ↓
Yes を選択
    ↓
自動 push
    ↓
KNOWLEDGE.md に記録
    ↓
GitHub Desktop で自動マージ
    ↓
Netlify が自動デプロイ
    ↓
完成！
```
