# 01 — プロジェクトセットアップ

## このプロンプトをコピペして使用

```
以下の情報を入力して、HP プロジェクトをセットアップしてください。

【クライアント情報】
- クライアント名: [伊東　尚悟]
- 業種: [政治家]
- サービス名: [道議会議員]
- ターゲット層: [釧路地域の有権者層（10代～70代）]

【参考サイト技術解析】
- CMS: WordPress
- SEOプラグイン: All in One SEO Pack 4.9.5.1
- 決済サービス: Stripe
- CDN: jsDelivr
- アイコンフォント: Font Awesome
- メタデータ: Open Graph Protocol
- メタデータ: Schema.org (JSON-LD)
- フィード: RSS

【作業内容】
1. genba-agent-prompts/HP作成/CLAUDE.md をコピー
   → [クライアント名] に置換

2. genba-agent-prompts/HP作成/SKILL.md をコピー
   → 参考サイト技術解析結果を反映
   → 政治家 HP 固有のセクション構成を提案

3. ★ 最適なセクション構成を決定
   （参考サイト技術 + ターゲット層から最適な構成を提案）

4. index.html の基本構造を作成
   → フォルダ構成を自動生成
   → package.json + scripts/ も自動生成

5. ★ GitHub リポジトリ用の .gitignore を作成

6. ★ README.md を作成（プロジェクト概要）
```
